# 📧 Respondedor Automático de E-mails — Claude API

Script em Python que gera respostas profissionais e personalizadas para e-mails de clientes, usando a API do Claude (Anthropic).

## Como usar
export ANTHROPIC_API_KEY="sua-chave-aqui"
python3 respondedor_emails.py
