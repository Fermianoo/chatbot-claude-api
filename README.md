# 🤖 Chatbot com IA — Claude API

Chatbot em Python com memória de conversa usando a API do Claude (Anthropic).

## O que faz
- Mantém o histórico completo da conversa
- Responde em português com personalidade definida
- Roda direto no terminal

## Tecnologias
- Python 3
- Anthropic Claude API
- claude-sonnet-4-6

## Como usar
```bash
export ANTHROPIC_API_KEY="sua-chave-aqui"
python3 chatbot.py
```
